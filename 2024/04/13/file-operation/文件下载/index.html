前置知识： 文件上传

利用 a 标签的 download 属性

相关概念

  1. createObjectURL 把 blob 对象的内存地址，以 url 形式给出

  2. msSaveblob兼容性考虑，IE 不支持 a 标签下载

  3. a标签的download属性表面该 a 标签的行为是下载，并说明文件名

流程

按 blob 格式请求接口返回二进制文件：

  1. 有 msSaveBlob —> 用msSaveBlob下载
  2. 无 msSaveBlob ：createObjectURL 创建文件本地 url —> 创建 a 标签 —> 创建 url 给到 a 标签的 href —> 设置 download —> 模拟点击 a 标签 —> 下载完成
