为什么绑定的事件使用的是pointer而不是mouse？

Pointer Events API 旨在统一处理不同输入设备（如鼠标、触摸、笔等）的事件。它综合了传统的鼠标事件（如 mousedown、mouseup、mousemove 等）和触摸事件（如 touchstart、touchend、touchmove 等）。使用 pointerdown 可以更好地支持各种输入设备，使得拖拽功能在多种设备上具有一致性
