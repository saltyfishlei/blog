什么是 BFF？

BFF（Backend for Frontend）的核心思想是为每种客户端体创建一个专门的后端服务，这些服务作为中间层，介于前端应用和核心业务逻辑（或数据源）之间

在前端开发常表现为通过 node 服务去直接返回已经渲染好数据的 html 模板。
