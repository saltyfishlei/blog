特殊类型与高级类型

any && unknow

  1. any 和 unkow 类型可以分配任何值
  2. unknown 相当于更安全的 any， unkown 类型只能赋值给 unkown 和 any

tips: 不可随意滥用 any 与 unkown 类型， 否则 ts 就失去了本身的意义

never（了解）

  1. never 代表永远不会出现
  2. never 类型的变量可以赋值给任何东西，但 never 类型不能有任何值
  3. 常用错误类型，或者永远没有结果的返回值

联合类型

满足其中一种类型, 使用管道符|定义

交叉类型

必须满足多个类型，一般用于两种不对的对象，使用连字符&

接口与 type

type

  1. type关键字用于给一个类型一个命名，可以用于各种类型的定义。

  2. 常用于基础类型和联合，交叉类型。方便复用

interface

  1. interface意思为接口，区别于 type，这是定义了一个可继承的接口
  2. 常用于函数，类，对象，数组的定义
